PTMCMCSampler
-------------
PTMCMC: Parallel Tempering Markov Chain Monte-Carlo sampler.

The aim of PTMCMC is to provide an adaptive jump proposal scheme where the user can addcustom jump proposals or use the default Adaptive Metropolis (AM) and Differential Evolution (DE) jump.
